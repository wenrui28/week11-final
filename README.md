# Week 11 Soccer Team Management App

Android project aligned with the Week 11 exercise requirements.

Main features:
- generic Repository<T extends SoccerEntity>
- common SoccerEntity interface
- Team, Player, and Match model classes with validation
- generic DataProvider<T>
- custom iterator implementation
- lambda-based filtering and sorting
- streams API usage
- TabLayout + ViewPager2 + fragments
- search and sorting UI
